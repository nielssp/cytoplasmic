      Create a two-way binding that modified values in both directions. The resulting cell is mutable and any modification of it will also result in an update of this cell.

      Type Parameters

      • T2

      Parameters

      • encode: (value: T) => T2

      Returns MutCell<T2>

      Example

      const b = a.bimap(x => x + 1, y => y - 1);
